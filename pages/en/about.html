Description of This Site

After Chart Cyanvas shut down and custom charts became temporarily inaccessible, I thought it would be a problem if there wasn’t a site where you could play charts offline. So, I created this site in about three days (both the code and the pages).
Some missing features are just the result of the "three-day quality."
I will continue adding more features over time.
