About SCP files

This is an explanation of SCP files.

What is an SCP file?

An SCP file is a Sonolus Collection Package file, which is a package file containing a Sonolus collection.

How to create an SCP file

The information on this page is based on Sonolus v1.0.0. Please note that the information may be outdated.
  1. If you have not yet created a collection, create one first.
  2. Launch Sonolus, select “Play,” and choose the collection you want to package.
  3. Select “Export.”
  4. Choose the mode. When uploading to this site, we recommend exporting in Full mode (although this will increase the file size...).
  5. Press the export button to start exporting.
  6. During the process, a screen will appear asking you to select a folder to save the file in. Choose any folder and save.
  7. When the export completes successfully, a message saying Package exported. will be displayed.
