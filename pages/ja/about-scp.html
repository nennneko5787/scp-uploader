SCPファイルについて

SCPファイルについての説明です。

SCPファイルとは

Sonolus ( ソノラス ) Collection ( コレクション ) Package ( パッケージ ) ファイルのことで、Sonolusのコレクションをパッケージ(ファイル)化したものです。

SCPファイルの作成方法

本ページの情報は、Sonolus v1.0.0時点のものです。情報が古くなっている場合があります。ご了承ください。
  1. まず、コレクションを作成していない場合は適当に作成します。
  2. Sonolus起動後、プレイを選択し、パッケージ化したいコレクションを選択します。
  3. エクスポートを選択します。
  4. モードを選択します。本サイトでアップロードする際はフルモードでエクスポートすることをおすすめしています。(その分ファイルサイズは増加しますが...)
  5. エクスポートボタンを押すと、エクスポートが始まります。
  6. 途中で保存するフォルダーを選択する画面が出てくるので、適当に選択して保存します。
  7. 正常にエクスポートが終わると、パッケージを書き出しました。と表示されます。
